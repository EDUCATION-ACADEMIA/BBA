A column in a relation / table is also known as ____

(A) field

(B) attribute

(C) feature

(D) ** all of the above

EXPLANATIONS BELOW

Concept note-1: -The columns in a table is a field and is also referred to as an attribute.

Concept note-2: -A table has records (rows) and fields (columns).

Concept note-3: -field. a. This is the correct answer because the columns in a table or relation are also known as a field. A field in a table contains a record related to a particular entity.

Concept note-4: -The primary key is a column, or set of columns, that allows each row in the table to be uniquely identified.

Concept note-5: -In the context of relational databases, a column is a set of data values, all of a single type, in a table. Columns define the data in a table, while rows populate data into the table. Most databases allow columns to contain complex data like images, whole documents or even video clips.
