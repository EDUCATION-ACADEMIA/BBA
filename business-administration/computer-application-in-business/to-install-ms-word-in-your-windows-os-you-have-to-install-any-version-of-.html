To Install MS Word in your Windows OS, you have to install any version of ____

(A) MS WORD

(B) ** MS OFFICE

(C) MS PAINT

(D) MS WINDOWS 7

EXPLANATIONS BELOW

Concept note-1: -We recommend the 32-bit version of Microsoft 365 for most users, because it’s more compatible with most other applications, especially 3rd-party add-ins. However, consider the 64-bit version, especially if you’re working with large blocks of information or graphics.

Concept note-2: -You can install and use more than one version of Office on a single computer. For example, you can install and use both Office 2013 and Office 2010 on the same computer. However, we do not recommend this.

Concept note-3: -Go to File > Help > Check for Updates. Choose Install Updates or Check for Updates.

Concept note-4: -You can only have one Version of Outlook running at a time. As has been said.. Office 365 Home Premium is a Virtualized app. You can actually run other versions of Office like 2010 at the same time just not both version of Outlook.
