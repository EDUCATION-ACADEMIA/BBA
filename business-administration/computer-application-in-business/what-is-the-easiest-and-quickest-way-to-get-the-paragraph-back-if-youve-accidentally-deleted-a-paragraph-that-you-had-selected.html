What is the easiest and quickest way to get the paragraph back if you’ve accidentally deleted a paragraph that you had selected?

(A) Exit the document without saving it.

(B) There is no easy way to do it.

(C) ** Use the Ctrl+Z keyboard shortcut.

(D) Use the keyboard to retype the paragraph.

EXPLANATIONS BELOW

Concept note-1: -Q. You’ve accidentally deleted a whole paragraph that you had selected; what is the easiest and quickest way to get the paragraph back? Use the Ctrl+Z keyboard shortcut. Use the keyboard to retype the paragraph.

Concept note-2: -The Google Docs Dictionary The quickest and easiest way to find synonyms in Google Docs is with the built-in Dictionary. Select the word you want a synonym for and either right-click and pick Define ‘[word]’ or click Tools > Dictionary from the menu.

Concept note-3: -What do you do if the spelling and grammar checker finds an error that is not really an error? Click Ignore.

Concept note-4: -A cursor is the position indicator on a computer display screen where a user can enter text. In an operating system with a graphical user interface (GUI), the cursor is also a visible and moving pointer that the user controls with a mouse, touch pad, or similar input device.

Concept note-5: -Cursor/ Insertion Point: The cursor (also known as the insertion point) is a flashing vertical bar on the screen that indicates where entered text or objects will be placed in the document.
