Large database that usually stores transactional records is ____

(A) Data storage

(B) Data records

(C) ** Data warehouse

(D) Database architecture

EXPLANATIONS BELOW

Concept note-1: -A data warehouse is a relational database that is designed for query and analysis rather than transaction processing. It usually contains historical data that is derived from transaction data, but it can include data from other sources.

Concept note-2: -A data warehouse is a database of a different kind: an OLAP (online analytical processing) database. A data warehouse exists as a layer on top of another database or databases (usually OLTP databases).

Concept note-3: -Databases efficiently store transactional data, making it available to end users and other systems. Data warehouses aggregate data from databases and other sources to create a unified repository that can serve as the basis for sophisticated reporting and analytics.

Concept note-4: -Transactional databases are a type of database management system (DBMS) optimized to read and write individual rows of data very quickly while protecting the integrity of the data they contain.
