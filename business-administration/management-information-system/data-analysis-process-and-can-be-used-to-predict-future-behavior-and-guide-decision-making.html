Concept note-1: -Data mining is a process of extracting useful information, patterns, and trends from raw data. Data analysis is a method that can be used to investigate, analyze, and demonstrate data to find useful information. The data mining output gives the data pattern.

Additionally, it helps you anticipate future customer needs so that you can plan ahead and stay one step ahead of the competition.
