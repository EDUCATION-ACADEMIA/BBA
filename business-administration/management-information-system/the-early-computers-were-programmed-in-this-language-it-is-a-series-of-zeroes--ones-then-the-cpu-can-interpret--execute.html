The early computers were programmed in this language. It is a series of zeroes & ones then the CPU can interpret & execute.

(A) ** SECOND GENERATION LANGUAGE (THE ASSEMBLERS)

(B) FIRST GENERATION LANGUAGES (MACHINE LANGUAGE)

(C) THIRD GENERATION LANGUAGES (3 GLs)

EXPLANATIONS BELOW

Concept note-1: -Machine language gives instructions as 0’s and 1’s and is the only language that the computer understands. Assembly language is more concise but still very cumbersome when programming.

Concept note-2: -Second Generation: By 1950, computers were being programmed in assembly language. As we saw earlier, one line in Assembly Language is translated into one machine instruction by a program called an assembler.

Concept note-3: -Computer Programming History: FORTRAN was the first computer programming language that was widely used. A Brief History of Computer Programming Languages: Computer code is the foundation of computers, enabling them to do the tasks humans need them to do.

Concept note-4: -Explanation: Machine language was used in creating the first computer programming system. In terms of functionality, this language is a binary digit system. The first computer systems primarily used binary numbers or bits to translate and understand processed programming.

Concept note-5: -That language of 1’s and 0’s is called binary. Computers speak in binary because of how they are built. A computer is nothing more than a vast collection of switches.
