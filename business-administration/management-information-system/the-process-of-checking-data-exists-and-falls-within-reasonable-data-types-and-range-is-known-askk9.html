The process of checking data exists and falls within reasonable data types and range is known as:(KK9)

(A) Input

(B) ** Validation

(C) Processing

(D) Output

EXPLANATIONS BELOW

Concept note-1: -Range Check A range check will verify whether input data falls within a predefined range.

Concept note-2: -Range Check. This determines whether or not supplied data falls within a specified range. Values outside of this range are invalid.

Concept note-3: -Field validation rules Use a field validation rule to check the value that you enter in a field when you leave the field.

Concept note-4: -Validation is an automatic computer check to ensure that the data entered is sensible and reasonable. It does not check the accuracy of data.
