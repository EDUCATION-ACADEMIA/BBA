A tickler file is a(n) ____

(A) alphabetical file of customers

(B) ** memory aid

(C) numeric file of customer numbers

(D) geographic file

EXPLANATIONS BELOW

Concept note-1: -A tickler file system is an organizational system that helps you remember tasks that need to be done or events that need to be attended. It is based on the principle of “tickling your memory” by having reminders of what needs to be done.

Concept note-2: -A tickler file is a paper or digital reminder system. It organizes a list of current or future tasks that need completed or tracked-either once or on a recurring basis.

Concept note-3: -A Tickler System is a form of reminder system designed specifically for lawyers. A tickler system is designed to remind lawyers of certain dates, such as compliance deadlines or filing deadlines that, if missed, could severely prejudice a lawyer’s client.

Concept note-4: -Tickler is defined as a written note to help you remember something in the future. An example of a tickler is a note that you keep on your mirror at home.

Concept note-5: -A tickler file stores reminders about important tasks that need to be done or attended to in the future. So it’s an effective way to stay organized and track what needs to be done. By writing out reminders in advance, a tickler file can help you stay ahead of deadlines and better manage future tasks or events.
