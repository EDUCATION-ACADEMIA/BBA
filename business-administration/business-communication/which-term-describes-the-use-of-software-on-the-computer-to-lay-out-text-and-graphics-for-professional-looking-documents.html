Which term describes the use of software on the computer to lay out text and graphics for professional-looking documents?

(A) visual literacy

(B) ** desktop publishing

(C) visual design

(D) graphic design

EXPLANATIONS BELOW

Concept note-1: -Desktop publishing (DTP) is the creation of documents using page layout software on a personal ("desktop") computer.

Concept note-2: -The five most popular desktop publishing packages-Adobe PageMaker, Adobe InDesign, QuarkXPress, Adobe FrameMaker, and Corel VENTURA-can help you do that.

Concept note-3: -1) Laser printer is very commonly used for desk-top publishing. 2) Desktop-publishing (DTP) software allows you to create page layouts that combine text and graphics. 3) A laser printer is a type of printer that uses a laser and electrical charge model instead of the traditional printing of ink onto paper.

Concept note-4: -Desktop publishing (DTP) is the use of personal computers to design books and booklets that are intended to be printed by ink jet or laser printers. The software that supports desktop publishing has a WYSIWYG graphical user interface (GUI) to make the set-up for publishing as easy as possible.

Concept note-5: -In this page you can discover 5 synonyms, antonyms, idiomatic expressions, and related words for desktop publishing, like: computer graphics design program, dtp, print formatter, text editor and word processor.
