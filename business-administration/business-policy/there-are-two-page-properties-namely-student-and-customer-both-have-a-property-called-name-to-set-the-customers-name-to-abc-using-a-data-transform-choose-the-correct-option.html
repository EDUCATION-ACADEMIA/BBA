There are two page properties, namely Student and Customer. Both have a property called Name. To set the Customer’s Name to “ABC” using a data transform, choose the correct option.

(A) .Name equal to “ABC”

(B) .Sudent.Name equal to “ABC”

(C) ** .Customer.Name equal to “ABC”

(D) .Name.Customer equal to “ABC”

EXPLANATIONS BELOW

Concept note-1: -An Entity Set is a set of entities of the same type that share the same properties, or attributes.

Concept note-2: -This relationship is also referred to as cardinality. For example, the following relationships are 1:N, or one-to-many: A student can register for multiple courses. A lecturer can deliver multiple courses.

Concept note-3: -Unary relationship: Association is maintained within a single entity. Recursive relationship: Relationship exists between occurrences of the same entity set. Binary relationship: Two entities are associated.

Concept note-4: -You define field value rules for those rule types that use labels or other text strings under 64 characters. For example, a property reference in a section can use a field value as the label text. Applications translated into other languages have a field value with the translation for each UI form property.
