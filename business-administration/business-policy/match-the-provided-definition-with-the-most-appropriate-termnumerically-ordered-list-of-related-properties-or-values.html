EXPLANATIONS BELOW

Concept note-1: -An ordered list typically is a numbered list of items. HTML 3.0 gives you the ability to control the sequence number-to continue where the previous list left off, or to start at a particular number.

Concept note-2: -The <ol> tag defines an ordered list. An ordered list can be numerical or alphabetical. The <li> tag is used to define each list item. Tip: Use CSS to style lists. Tip: For unordered list, use the <ul> tag.

Concept note-3: -Working with HTML Lists Unordered list-Used to create a list of related items, in no particular order. Ordered list-Used to create a list of related items, in a specific order. Description list-Used to create a list of terms and their descriptions.

Concept note-4: -The HTML <ol> tag defines an ordered list. An ordered list can be numerical or alphabetical.
