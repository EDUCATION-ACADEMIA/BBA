A method to analyze multidimensional data from many different perspectives, enabling users to identify issues and opportunities as well as perform trend analysis

(A) Data Cube

(B) Word Cloud

(C) Drill-down Analysis

(D) ** Online Analytical Processing

EXPLANATIONS BELOW

Concept note-1: -Online Analytical Processing: Method used to analyze multidimensional data from many different perspectives (enables users to identify issues and opportunities and perform trend analysis).

Concept note-2: -Relational OLAP (ROLAP) is OLAP that performs dynamic multidimensional analysis of data stored in a relational database.

Concept note-3: -Dimensional Analysis OLAP can support aggregation of cumulative data (e.g., sums, averages, minimums, maximums) or numeric data (such as sales, number of employees, number of products produced, inventories) presented in a way that allows for interactive “drill-down, ” or successive detail analysis and exposition.

Concept note-4: -Multidimensional data analysis is a method that, according to the management information demands, users observe the data from several angles of the real world and make a corresponding processing so as to get some useful information. Multidimensional data analysis cannot do without the foundation of data warehouse.

Concept note-5: -Data retrieval An OLAP server returns new data to Excel every time that you change the layout of the report.
