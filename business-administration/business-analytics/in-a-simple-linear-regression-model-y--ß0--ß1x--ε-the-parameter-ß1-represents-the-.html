Concept note-1: -The regression model is given by: Y = + X + where is the y intercept (the value of Y where X = 0), is the slope of the line, and is a random error term. It may also be given as: Y = 0 + 1X + where 0 is the y intercept, 1 is the slope of the line, and is a random error term.

Concept note-2: -True The slope of the simple linear regression equation represents the average change in the value of the dependent variable per unit change in the independent variable (X).

Concept note-3: -To calculate slope for a regression line, you’ll need to divide the standard deviation of y values by the standard deviation of x values and then multiply this by the correlation between x and y. The slope can be negative, which would show a line going downhill rather than upwards.

Concept note-4: -Answer and Explanation: According to the definition of the equation of regression analysis, the Y-intercept represents the predicted value of Y when X = 0.
