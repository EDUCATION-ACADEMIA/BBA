Can usually be counted in a finite matter.

(A) Continuous Variable

(B) Quantitative

(C) ** Discrete Variable

(D) Volume

EXPLANATIONS BELOW

Concept note-1: -A discrete variable is a variable that takes on distinct, countable values. In theory, you should always be able to count the values of a discrete variable.

Concept note-2: -Discrete Data: Discrete data is a type of numerical data with countable elements. I.e they have a one-to-one mapping with natural numbers. Discrete data can either be countably finite or countably infinite.

Concept note-3: -Discrete random variables can only take on a finite number of values. For example, the outcome of rolling a die is a discrete random variable, as it can only land on one of six possible numbers. Continuous random variables, on the other hand, can take on any value in a given interval.

Concept note-4: -Discrete data is a numerical type of data that includes whole, concrete numbers with specific and fixed data values determined by counting. Continuous data includes complex numbers and varying data values measured over a particular time interval.
