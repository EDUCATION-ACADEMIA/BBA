Let A be a non-singular matrix of order (3 × 3). Then |adj A| is equal to ____

(A) |A|

(B) ** |A|2

(C) |A|3

(D) 3|A|

EXPLANATIONS BELOW

Concept note-1: -The rank of the non-singular matrix of order 3/3 is 3. A non-singular matrix is a square matrix with non zero determinant.

Concept note-2: -For a Singular matrix, the determinant value has to be equal to 0, i.e. |A| = 0. As the determinant is equal to 0, hence it is a Singular Matrix.

Concept note-3: -And we know that if A is a matrix of order 3 then $|adjA| = |A|^n-1$ where n is the order of the matrix. Here order is 3 so n = 3. Hence, the value of K is 2.

Concept note-4: -Detailed Solution Calculation: From the properties of the determinants, we know that |KA| = Kn |A|, where n is the order of the determinant. Here, n = 3, therefore, the answer is K3 |A|.
