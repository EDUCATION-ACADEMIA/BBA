(A) The abbreviation is M

(B) ** The formula is to calculate the Markdown is, Markdown=Old Selling Price-New Selling Price

(C) The formula is to calculate the Markdown is, Markdown=New Selling Price-Old Selling Price

(D) ** Is the reduction in the selling price of merchandise due to many reasons such as to increase sales, to face stiff competition, to encourage purchase in bulk, to dispose old, damaged/obsolete stocks

(E) ** The abbreviation is MD

EXPLANATIONS BELOW

Concept note-1: -Text files created with Markdown language dialects is saved with . md or . MARKDOWN file extension. MD files are saved in plain text format that uses Markdown language which also includes inline text symbols, defining how a text can be formatted such as indentations, table formatting, fonts, and headers.

Concept note-2: -Markdown is a lightweight markup language that describes how text should look on a page. HTML is another example of a markup language. Markdown is a style of writing documents that makes it easy to define what the content should look like. It describes headers, text styles, links, lists and so much more.

Concept note-3: -md file extension (also written as . markdown) stands for “Markdown documentation”. Markdown is a lightweight markup language, and each MD file is written in a particular “flavor” of Markdown. It can be helpful to think of the different flavors as being like different dialects of one language.

Concept note-4: -Markdown is an easy-to-use markup language that is used with plain text to add formatting elements (headings, bulleted lists, URLs) to plain text without the use of a formal text editor or the use of HTML tags. Markdown is device agnostic and displays the writing format consistently across device types.

Concept note-5: -When you create a repository or a project, GitHub gives you the option of a default readme. The default readme file contains the repository name and some basic instructions. The file format is ‘md’, which stands for Markdown documentation. It is a lightweight markup language that can be easily converted to text.
