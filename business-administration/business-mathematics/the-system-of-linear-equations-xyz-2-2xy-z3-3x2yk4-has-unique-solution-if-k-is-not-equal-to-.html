The system of linear equations x+y+z =2, 2x+y-z=3, 3x+2y+k=4 has unique solution, if k is not equal to ____

(A) 4

(B) ** 0

(C) -4

(D) 1

EXPLANATIONS BELOW

Concept note-1: -In a set of linear simultaneous equations, a unique solution exists if and only if, (a) the number of unknowns and the number of equations are equal, (b) all equations are consistent, and (c) there is no linear dependence between any two or more equations, that is, all equations are independent.

Concept note-2: -A system of linear equations a 1 x + b 1 y = 0 a 2 x + b 2 y = 0 has a unique solution, if a 1 a 2 ≠ b 1 b 2 .

Concept note-3: -A system of linear equations has no solution when the graphs are parallel.

Concept note-4: -(1) No solution occurs only for k = 1.
