A variable is a characteristic that takes on two or more values whereas, an attribute is a specific value on a variable (qualitative).

(A) ** TRUE

(B) FALSE

EXPLANATIONS BELOW

Concept note-1: -A categorical variable that can take on exactly two values is termed a binary variable or a dichotomous variable; an important special case is the Bernoulli variable.

Concept note-2: -A categorical variable (also called qualitative variable) refers to a characteristic that canâ€™t be quantifiable. Categorical variables can be either nominal or ordinal.

Concept note-3: -Data collected about a numeric variable will always be quantitative and data collected about a categorical variable will always be qualitative. Therefore, you can identify the type of data, prior to collection, based on whether the variable is numeric or categorical.

Concept note-4: -Characteristics of Variable 1. Capable of assuming several values representing a certain category. 2. Values that may arise from counting and or from measurement.
