Quantitative variable are

(A) ** Measurable characteristic

(B) ** Able to count

(C) ** Can be continuous

(D) Dichotomous variable

EXPLANATIONS BELOW

Concept note-1: -A numeric variable (also called quantitative variable) is a quantifiable characteristic whose values are numbers (except numbers which are codes standing up for categories). Numeric variables may be either continuous or discrete.

Concept note-2: -Quantitative variables may be discrete or continuous. Discrete variables can only take on a limited number of values (e.g., only whole numbers) while continuous variables can take on any value and any value between two values (e.g., out to an infinite number of decimal places).

Concept note-3: -Variables that can only take on a finite number of values are called “discrete variables.” All qualitative variables are discrete.

Concept note-4: -On the other hand, quantitative continuous variables are variables for which the values are not countable and have an infinite number of possibilities. For example: Age. Weight.

Concept note-5: -A continuous variable is a type of quantitative variable consisting of numerical values that can be measured but not counted, because there are infinitely many values between 1 measurement and another. Example: Cholesterol level measured in mg/dl.
