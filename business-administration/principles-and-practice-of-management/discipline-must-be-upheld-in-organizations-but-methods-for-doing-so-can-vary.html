Discipline must be upheld in organizations, but methods for doing so can vary.

(A) ** Discipline

(B) Centralization

(C) Authority

EXPLANATIONS BELOW

Concept note-1: -Discipline must be upheld in organizations, but methods for doing so can vary. Employees should have only one direct supervisor. Teams with the same objective should be working under the direction of one manager, using one plan. This will ensure that action is properly coordinated.

Concept note-2: -Discipline must be upheld in organizations, but methods for doing so can vary. Unity of Command.

Concept note-3: -Discipline This principle states that discipline is required for any organization to run effectively. In order to have disciplined employees, managers need to build a culture of mutual respect. There should be a set of organizational rules, philosophies, and structures in place that should be met by everyone.

Concept note-4: -Discipline is defined as obedience towards organizational rules and employment agreements that are necessary for the working of the organization. According to Fayol, discipline requires good superiors at all levels, clear and fair agreements, and judicious application of penalties.
